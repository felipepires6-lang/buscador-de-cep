# 📍 Buscador de CEP Residencial

Um aplicativo web interativo e responsivo desenvolvido para facilitar a busca de endereços no território brasileiro. O projeto consome dados da API pública ViaCEP em tempo real.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando conceitos modernos de desenvolvimento web tradicional (Vanilla):
- **HTML5:** Estruturação semântica da interface.
- **CSS3:** Estilização moderna e layout responsivo com gradientes e animações de transição.
- **JavaScript (ES6):** Manipulação assíncrona de dados, controle de eventos do DOM e tratamento de erros através de estruturas try/catch.

## 💡 Funcionalidades Técnicas Aprendidas

Durante o desenvolvimento deste projeto, pratiquei os seguintes conceitos essenciais para o mercado:
1. **Consumo de APIs externas:** Uso prático do método assíncrono `fetch` integrado com `async/await`.
2. **Manipulação dinâmica do DOM:** Atualização de elementos textuais e visuais da tela em tempo real com base na interação do usuário.
3. **Tratamento de Exceções:** Criação de validações de entrada (comprimento do campo) e blocos de captura de erros (`try/catch`) para evitar falhas silenciosas na aplicação.

## 🔧 Como Rodar o Projeto Localmente

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente no seu navegador de preferência.
3. Insira um CEP válido com 8 dígitos (apenas números) e clique em "Buscar Endereço".
